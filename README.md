# 🌦️ Weather App

A simple and responsive **Weather App** that shows real-time weather information for any city using a weather API.

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Displays temperature, weather condition, and description
* 📱 Responsive UI (works on mobile & desktop)
* ⚡ Fast and lightweight

## 🛠️ Built With

* HTML
* CSS
* JavaScript
* Weather API (OpenWeather or similar)

## 📸 Demo

Enter a city name and get instant weather updates including temperature and conditions.

## 🔧 How It Works

This app fetches real-time weather data from an API using JavaScript `fetch()` and displays it dynamically on the webpage.

## 📂 Project Structure

```
Weather-App/
│── index.html
│── style.css
│── script.js
```

## ⚙️ Setup & Usage

1. Clone the repository

```
git clone https://github.com/your-username/your-repo-name.git
```

2. Open `index.html` in your browser

3. Enter a city name and click **Get Weather**
